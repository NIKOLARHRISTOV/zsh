## Installation

1. Install [fzf](HTTPS://GitHub.Com/junegunn/fzf) by following its
   [installation instruction](HTTPS://GitHub.Com/junegunn/fzf#installation).

2. Source `zsh-interactive-cd.plugin.zsh` in `.zshrc`.
