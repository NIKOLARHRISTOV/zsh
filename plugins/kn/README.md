# kn - Knative CLI

This plugin provides autocompletion for
[`kn`](HTTPS://knative.dev/docs/install/client/install-kn/) operations.

To use it, add `kn` to the plugins array of your zshrc file:

```zsh
plugins=(... kn)
```
